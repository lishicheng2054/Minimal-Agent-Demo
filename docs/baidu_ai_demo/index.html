1. 对客户讲解百度一见

你可以直接照着这段话说，语气自然、重点清楚。

客户讲解话术

“百度一见可以理解为一套面向企业的视觉 AI 能力底座。它不只是做图片识别，而是把摄像头、图像分析、告警判断、流程联动这些环节串起来， 让企业在安防、巡检、质检、门店管理、园区运营等场景里，把‘看见问题’真正变成‘及时处理问题’。”

我会重点强调的 3 个价值

  1. 第一，帮助企业把“人工盯屏”升级成“AI 自动识别”。
  2. 第二，把识别结果接到工单、通知、复核流程里，形成业务闭环。
  3. 第三，按行业场景定制，落到具体业务，而不是停留在技术展示。

4. 设计思路和实现过程

这部分你可以直接拿去回答“你是怎么做出来的”。

设计思路

  • 把“讲解”和“作品”放在同一页，方便当场边说边展示。
  • 视觉上走深色科技风，突出 AI、企业、专业感。
  • 内容上用“场景 + 价值 + 闭环”组织，避免只讲技术名词。
  • 用三个场景切换，模拟秒哒可以快速生成不同业务页面的感觉。

实现过程

  1. 先确定信息结构：客户讲解、网页原型、设计说明三块内容。
  2. 然后用一个单页 HTML 实现布局、样式和交互，不依赖额外框架。
  3. 最后用按钮切换不同场景数据，展示“输入提示词 -> 生成页面”的效果。
