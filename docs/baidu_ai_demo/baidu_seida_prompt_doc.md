# 百度秒哒生成提示词文档

## 文档目的

这份文档用于在百度秒哒中生成一个可直接展示的企业级网页原型，主题为“智慧园区巡检看板”。

你可以直接复制下面的提示词到百度秒哒中使用，也可以根据实际行业场景替换为门店、工厂、楼宇、仓储等版本。

---

## 一、完整提示词

```text
请生成一个企业级单页网页原型，主题为“智慧园区巡检看板”。

整体要求：
- 风格：深色科技风、企业级、简洁高级、有 AI 感
- 页面语言：中文
- 布局：适配电脑和手机，采用响应式设计
- 结构：一页式展示，内容清晰，适合给客户演示
- 目标：突出“AI 识别 -> 告警 -> 处理 -> 复核”的业务闭环
- 页面气质：不要像普通营销页，要更像一个真实企业内部系统或客户演示系统
- 组件风格：卡片化布局、信息层级分明、局部高亮、数据仪表感、轻量动效
- 交互要求：支持场景切换、按钮反馈、模块展开、数据联动、状态标签变化
- 内容要求：模块要完整，避免只放标题和空卡片，必须体现业务场景和管理动作

页面内容要求：

1. 顶部 Hero 区
- 主标题：智慧园区巡检看板
- 副标题：AI 自动识别异常，快速生成告警与工单，帮助企业实现巡检闭环与持续优化
- 主按钮文案：立即查看演示
- 次按钮文案：切换场景
- Hero 区要有明显视觉层次，可以加入渐变光效、科技感背景、数据浮层或简洁插图
- Hero 区补充信息：
  - 当前巡检范围
  - 今日异常总数
  - 当前告警等级
  - 最新更新时间

2. 数据总览区
展示 4 个 KPI 卡片，不只是数值，还要带趋势或辅助说明：
- 今日巡检次数：128
- AI 识别异常：14
- 待办工单：9
- 已完成复核：75

每个 KPI 卡片最好带一行辅助说明，例如：
- “较昨日提升 12%”
- “高危告警 3 条”

3. 异常分布与趋势区
建议增加一个图表模块或趋势模块，展示：
- 本周告警趋势
- 异常类型占比
- 风险等级分布
- 告警处理进度

可以用折线图、柱状图、环形图、进度条或者简化的趋势卡片来表现。

4. 异常告警区
展示一个告警列表，至少 4 条，内容尽量贴近真实企业场景，例如：
- 设备离线
- 区域人员聚集
- 通道堆物
- 夜间异常移动

每条告警建议补充：
- 所属区域
- 风险等级
- 触发时间
- 当前处理状态
- 处理人或责任人

5. 巡检流程区
用 3 个步骤卡片展示业务闭环：
- 识别
- 判断
- 联动

每个步骤卡片最好说明：
- 这一阶段在做什么
- 输入是什么
- 输出是什么
- 对业务有什么价值

6. 智能助手建议区
展示一段 AI 建议文案，例如：
- 建议优先处理 3 号楼南侧通道堆物问题
- 建议将高优先级告警同步给值班负责人

智能助手区还可以增加：
- 推荐处理顺序
- 风险等级解释
- 最近 3 条处理建议
- 一键生成工单按钮

7. 场景切换
提供 3 个可切换场景标签：
- 园区巡检
- 门店巡检
- 工厂质检

要求切换后，页面中的标题、KPI、告警内容和流程文案随场景变化。

8. 业务管理模块
建议再补充以下模块中的至少 2 个：
- 巡检任务列表
- 工单处理状态
- 负责人待办
- 告警详情抽屉
- 设备/区域概览
- 最近处理记录

9. 交互功能
建议实现以下至少 3 种交互：
- 点击切换场景，整页内容联动变化
- 点击告警项，展示详情或状态
- 点击按钮，模拟“生成页面”“生成工单”或“查看详情”
- 鼠标悬停卡片时显示轻微高亮
- 状态标签实时变化，例如“已生成”“处理中”“待复核”

设计要求：
- 视觉上要有层次感
- 卡片圆角，阴影柔和
- 科技感背景，可用渐变和光晕效果
- 文字要清晰，适合汇报演示
- 页面整体不要太复杂，重点突出“AI 识别 -> 告警 -> 处理”的闭环
- 版面要足够丰富，但不能堆满无意义装饰
- 保持信息密度适中，让客户能快速理解业务价值

输出要求：
- 直接生成完整网页
- 页面内容要真实、专业、适合企业客户展示
- 不要放占位符文字
- 生成后尽量可直接打开查看
- 优先生成可以演示的完整结构，而不是只有视觉效果的静态页面
```

---

## 二、如何使用

1. 打开百度秒哒。
2. 新建一个网页生成任务。
3. 把上面的完整提示词整体复制进去。
4. 先生成“园区巡检”版本，确认整体风格和结构。
5. 再切换到“门店巡检”和“工厂质检”，检查场景变化是否生效。

---

## 三、推荐的输出结构

建议页面最终包含以下部分：

- 顶部总览
- 关键数据卡片
- 异常告警列表
- 业务流程闭环
- AI 建议区
- 场景切换区

这样做的好处是：

- 客户一眼能看懂业务价值
- 页面更适合现场演示
- 结构清晰，便于后续扩展

---

## 四、可替换行业版本

如果你想做不同场景，可以直接替换主题和内容：

- 园区巡检
- 门店巡检
- 工厂质检
- 仓储巡检
- 楼宇安防

只需要把提示词里的主标题、KPI、告警列表和流程文案替换掉即可。
