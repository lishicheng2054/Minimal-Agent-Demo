# 百度一见 × 百度秒哒 演示说明

## 这个文件夹里有什么

- `index.html`：演示页入口，打开即可查看

## 这次交付的内容

1. 模拟对客户讲解“百度一见”
2. 用“百度秒哒”思路做一个简单网页原型
3. 补充设计思路和实现过程，方便现场演示

## 页面结构

- 顶部：项目总览
- 中间第一块：百度一见客户讲解
- 中间第二块：秒哒风格的页面原型
- 底部：设计思路和实现过程

## 演示建议

1. 先讲“百度一见”的客户价值
2. 再切到“园区巡检看板”原型
3. 通过“园区 / 门店 / 工厂”切换展示生成效果
4. 最后讲页面设计思路和落地方式

## 设计思路

- 用深色科技风，突出 AI 和企业级产品感
- 用一页式结构，把“讲解”和“作品”放在一起
- 用场景切换，模拟 AI 应用生成工具的快速出稿能力
- 用 KPI、告警、流程卡片表达业务闭环

## 实现方式

- 使用单个 HTML 文件完成布局、样式和交互
- 通过 JavaScript 切换三个业务场景
- 不依赖额外框架，打开文件即可展示
